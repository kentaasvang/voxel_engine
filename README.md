## 🔧 Building and Running

**Install dependecies**

```sh
pip install -r requirements.txt
``` 

**Install osmesa-library**
```sh
sudo apt install libosmesa6

```

**Install freeglut3-dev**
```sh
sudo apt-get install freeglut3-dev
```

**Run**
```sh
./main.py
```
